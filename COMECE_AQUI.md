# 🎯 COMECE AQUI

Bem-vindo! Este é seu **App Presença Digital**.

## 📱 O que é?

Um app para você marcar presença dos alunos **pelo celular**, salvando automaticamente nos arquivos Excel dos diários.

---

## ⚡ Início Rápido (3 passos)

### 1️⃣ Testar Localmente (seu computador)

```bash
cd "D:\1. PROJETOS\ESCOLA\2026\DIÁRIOS\app-presenca"
npm install
npm start
```

Pronto! Acesse: `http://localhost:3000` no seu navegador

### 2️⃣ Usar na Escola (com WiFi)

- Leve seu notebook para a sala
- Mesmo WiFi no celular
- Acesse: `http://seu-ip:3000` (ex: `http://192.168.1.100:3000`)

### 3️⃣ Usar Online (Replit - Recomendado)

Veja: **GUIA_REPLIT.md** para colocar na nuvem (funciona de qualquer lugar)

## ✨ Funcionalidades

✅ Seleciona a turma na tela inicial  
✅ Marca presença dos alunos (P ou F)  
✅ Anota conteúdo da aula  
✅ Adiciona novos alunos  
✅ Salva automaticamente no Excel  
✅ Funciona em múltiplas turmas  
✅ Interface mobile-friendly

## 🆘 Problemas?

- **Node.js não instalado:** https://nodejs.org/
- **Erro de módulos:** Rode `npm install` novamente
- **Arquivo não encontrado:** Verifique nome dos diários (devem ter "DIÁRIO")
- **Replit não funciona:** Veja GUIA_REPLIT.md passo a passo
